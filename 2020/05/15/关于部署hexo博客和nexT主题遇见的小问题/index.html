关于部署hexo博客和nexT主题遇见的小问题

发表于 2020-05-15

安装npm中国镜像

  众所周知的一些原因，npm在中国的下载速度非常之慢，这里可以下载npm的某宝镜像来解放限速。打开terminal执行以下指令即可。

npm install -g cnpm --registry=http://registry.npm.taobao.org


更改默认作者姓名

   进入blog的配置文件（此处为_config.yml）后在site中更改author。需要注意的是更改完成后需要重启hexo serve。

hexo cl && hexo g && hexo s
